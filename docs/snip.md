## snip

Quick Snip is a tool to save your thought snippets

### Synopsis

A fast and flexible cli tool to save your thought snippets and read them again

```
snip [flags]
```

### Options

```
      --config string   config file (default is $HOME/.snip.yaml)
      --debug           enable debug mode
  -h, --help            help for snip
```

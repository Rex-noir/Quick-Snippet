## snip add

Add a new snippet

### Synopsis

Add a new snippet directly or interactively if -i flag is provided.

```
snip add [title] [body] [flags]
```

### Options

```
  -h, --help          help for add
  -i, --interactive   Run in interactive mode to enter snippet details
```

### Options inherited from parent commands

```
      --config string   config file (default is $HOME/.config/snip/config.yaml)
```
